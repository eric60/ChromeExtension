Set your Personal Youtube Key

You can create a free Youtube key for your own personal use so the extension knows what videos to block.
Google may ask for a payment method but you can remove it after you create your key.

Setup (6 minutes)

  1. Go to Google's Cloud Platform (GCP) and create a free account using your google account
  2. Create a new project
  3. Click the left top navigation menu and go to APIs and Services and create a new Youtube Data API
    1. You can create up to 10 keys for free if you run out, but this is unlikely.
  4. Click credentials in the left hand menu and click Create credentials on the page then click API key in the drop down
  5. Copy and paste your new personal API key in the input below to register it to the extension. It will be a long text key like this: AIzaSyAeebo7DlkB6YyCem51Lq9AOAmFG
  6. Set your API key then press the Activate button on the extension. Refresh the video if it doesn't work right away. Double check your key is correct if doesn't work after 5 refreshes.
  7. You can remove your billing account since YouTube Data doesn't cost any money
