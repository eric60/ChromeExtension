Set your Personal Youtube Key and Never Run Out

You can create a Youtube key for your own personal use so the extension knows what videos to block or you can use the shared key. The shared key will reach its limit if more than 5 people are using it, so please create a key and you won't run out of calls.

Instructions to Setup a Free Key

  1. Go to Google's Cloud Platform (GCP) and create a free account
  2. Create a new project
  3. Click the left top navigation menu and go to APIs and Services and create a new Youtube Data API
  4. Click credentials in the left hand menu and click Create credentials on the page then click API key in the drop down
  5. Copy and paste your new personal API key in the input below to register it to the extension. It will be a long text key like this: AIzaSyAeebo7DlkB6YyCem51Lq9AOAmFG

Or leave blank and save to use default key. WARNING: Key will reach its limit and not work if there are more than 5 users so please set your own key.

2) Finally press activate in the extension. It might take 2-3 videos until the key registers so refresh the video or try another video until it works. Ignore the warnings of an invalid key unless it still doesn't work after 5 different Youtube videos. Then double check your key.
